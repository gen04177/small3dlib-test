# small3dlib-test

`hbldr small3dlib-modelViewer.elf`

## Controls:

- **Left Stick**:       rotate model
- **Right Stick**:      zoom (FOV) in/out
- **Cross**:            next model
- **Circle**:           toggle wireframe
- **Square**:           toggle light
- **Triangle**:         toggle fog
- **L1**:               change backface culling
- **R1**:               toggle noise
